# Date Girls Easy project brief

Date Girls Easy is a content publisher for men who want better results from dating sites and dating apps.

The site should help users:

1. Choose the right dating site
2. Avoid wasting money
3. Improve profile quality
4. Send better first messages
5. Move toward real dates with less friction

The site should monetize through:

1. Affiliate links
2. Ebook sales
3. Email capture

The site should not include:

1. Join pages
2. Signup flows
3. User profiles
4. Member dashboards
5. Chat systems
6. Matchmaking features

Core site sections:

1. Reviews
2. Comparisons
3. Guides
4. Ebooks
5. About
6. Disclosure
7. Contact

Primary audience:
Men looking for clear, practical dating advice and honest dating site reviews.

Design direction:
Dark, clean, trustworthy, mobile first, conversion oriented, app friendly.

SEO and AI answer goals:

1. Strong page structure
2. Clear section headings
3. Direct answers near top of page
4. Internal linking across topical clusters
5. Structured data where useful
6. Clear editorial positioning
