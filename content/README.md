# Content rules

## Folder purpose

- content/ebooks = raw source library for research, summaries, metadata extraction, and future curation
- content/ebooks-live = ebooks approved to be surfaced on the public site
- content/guides = guide drafts and guide source material
- content/reviews = review notes and review source material

## Build rules for Codex

1. Do not automatically publish, link, or surface every file found in content/ebooks
2. Treat content/ebooks as a private source library unless a file is intentionally copied or represented inside content/ebooks-live
3. Only build public ebook pages, ebook listings, download links, or storefront blocks from content/ebooks-live
4. It is allowed to use content/ebooks as source material for summaries, categorization, metadata, and content planning
5. Keep site positioning focused on dating education, reviews, comparisons, guides, and ebooks for men
6. Do not position the site as a dating app, matchmaking service, or member platform
