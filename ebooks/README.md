# Ebook Library Structure

This folder is the public-facing home for the future Date Girls Easy HTML guide library.

Publishing model:

- Source PDFs in `content/ebooks` are private working material
- Public ebook content will be rewritten into original long-form HTML guides
- No raw PDF links should be surfaced on the live site
- Future guides should live under topic-based folders inside `ebooks/`
